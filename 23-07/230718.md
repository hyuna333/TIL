# 블로그 제작 & 기본 문법

## React 쓰는 이유
1. Single Page Application 만들 때 사용
2. 유저가 가장 많다
3. html을 함수, array, object에 넣어서 사용할 수 있어서 재사용 편리
4. 같은 문법으로 앱개발 가능(React Native)

## 개발환경 및 프로젝트 설치
- node.js, vscode 사용
- `npx create-react-app blog`
- node_modules : 프로젝트 구동에 필요한 모든 라이브러리의 소스코드를 모아둔 곳
- public : static 파일 보관함
- App.js : main page
- pakage.json : 프로젝트 정보
