# 블로그 제작 & 기본 문법 3

## class를 이용한 옛날 React 문법
- class를 이용해 component 생성
```js
// 기본 틀
class Modal2 extends React.Component {
  constructor() {
    super()
  }
  render(){
    return (
      <div>안녕</div>
    )
  }
} 

// state, props 사용
class Modal2 extends React.Component {
  constructor(props) {
    super(props);
    this.state = {
      name : 'kim',
      age : 20
    }
  }
  render(){
    return (
      <div>안녕 {this.state.name} {this.props}
        {/* state 변경 button */}
        <button onClick={()=>{
          this.setState({age : 21})
        }}>버튼</button>
      </div>
    )
  }
}
```
- 기존 state를 갈아치워주는 것이 아니라 기존 state에 변경사항만 반영해줌
